    Basic Terminologies 
    Network
    A network is a system 
of connected devices  
that allow users to 
communicate and share 
resources.Networks can
be local, metropolitan, 
wide, or personal. 
    
Nodes
    A node is a device that
connects to a network and
can send, receive,or 
process data. 
    
Protocols
    A protocol  is a set of
rules that govern how 
devices communicate with
each other.Protocols 
allow devices to communicate
even if they have different 
software, hardware, or 
internal processes. 
    
Topology
    Topology refers to the physical
or logical arrangement of
devices and connections.
It defines how the devices 
are connected and how 
data is transmitted 
between them. 
    
IP address 
    An IP address is a unique 
number that identifies a 
device on the internet.
It's like a phone number
that allows devices to 
communicate with each other. 
    
Subnet 
    A subnet is a smaller network 
within a larger network.
Subnets are created by
dividing an IP network
into multiple smaller 
networks.
