    OSI Model (Open Systems Interconnection) 
    Physical layer
    The physical layer is the first layer 
in the OSI model of networking. 
It is responsible for the actual 
transmission of raw data over 
a physical medium, such as cables
or wireless signals.
    
Data link layer
    The Data Link Layer is the second layer
in the OSI  model. It plays a 
crucial role in ensuring reliable 
communication between two devices 
over the same network, using the 
physical mediumto transfer data. 
    
Network layer
    The Network Layer is the third layer 
of the OSI  model, which is a 
conceptual framework used to
understandand standardize the 
functions of communication systems.
    
Transport layer
    The Transport Layer is the fourth layer
of the OSI  model and plays 
a critical role in ensuring reliable
data transfer between devices 
over a network.
    
Session layer
    The Session Layer is the fifth layer 
for managing sessions,or the continuous
exchange of data between two devices
over a network. 
    
Presentation layer
    The Presentation Layer is the sixth layer 
of the OSI  model,and it is primarily
responsible for ensuring that data 
is properly formatted, encrypted,
and presented in a way that the 
application layer can understand.
    
Application layer
    The Application Layer is the seventh layer 
of the OSI model. It is responsible for 
providing network services directly to 
end-users and supporting the
applications that users interact with.
